# Spark-like particle grnerator/simulation made in Pygame by jromanki

## Installation

### Requirements
    - Python3
    - Pygame

### How to run
    - Download the repository
    - Open downloaded folder in cmd/powershell/terminal emulator
    - Run: "python pygame-sparks.py"
